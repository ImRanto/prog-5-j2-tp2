# Système de Location - Prog 5 J2 TP2

Un système de gestion de location d'objets simple développé en Java

## Fonctionnalités

-  Gestion des objets localisables (Voitures, Maisons, Ordinateurs)
-  Gestion des locataires (Particuliers, Entreprises)
-  Système de location avec suivi
-  Architecture modulaire (Entities, Services)
-  Formatage automatique avec Google Java Format (lance ``format.bat/format.sh``)
-  CI/CD avec GitHub Actions

## Tech utilisé

- **Java 24** - Langage de programmation
- **Maven** - Gestion des dépendances
- **Google Java Format** - Formatage du code
- **GitHub Actions** - Intégration continue
